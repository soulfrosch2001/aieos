# Security Council — Output

## Output Format
Every session produces minutes using
[../../09-templates/meeting-minutes-template.md](../../09-templates/meeting-minutes-template.md):
Consensus · Remaining Concerns · Risks · Alternatives Considered ·
Recommendation · Implementation Strategy · Owners & Next Steps.

Risk-acceptance decisions must name the accepting authority and record any
[Chief Auditor](../../01-executive/chief-auditor/) veto or override. The finished
record is appended to
[../../07-memory/meeting-history.md](../../07-memory/meeting-history.md).

## Updates To Memory
- **Always:** [../../07-memory/meeting-history.md](../../07-memory/meeting-history.md)
  — the decision and who accepted the residual risk.
- **If risk is accepted or deferred:**
  [../../07-memory/technical-debt.md](../../07-memory/technical-debt.md).
- **If an incident yields a lesson:**
  [../../07-memory/lessons-learned.md](../../07-memory/lessons-learned.md).

Memory is append-mostly: add corrections, never rewrite history. See
[../../00-company/conventions.md](../../00-company/conventions.md).
