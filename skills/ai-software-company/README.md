# AI Software Company

An **operating system for software engineering** that runs inside Claude Code. Not a
collection of prompts — a self-coordinating organization of AI specialists that plans,
debates, and reviews **before** it writes code.

> Copy this folder into any repository to instantly get: executive management, technical
> leadership, engineering & product teams, quality and security review, architecture
> governance, shared memory, workflow automation, and continuous improvement.

## How It Works
```
Request → Orchestrator → Tier (T0–T4) → Council? → Plan → Gate → Build → Review → Memory
```
See [00-company/core-flow.md](00-company/core-flow.md) and

## Two Rules That Define Everything
- **Coordinate before you code.** Implementation is step six, not step one.
- **Files target ≤50 lines** (exceed only when truly indivisible). One agent = one folder.
  See [00-company/conventions.md](00-company/conventions.md).
