# 07-memory/minutes — Full Council Minutes

Status: stable

Each council / T2+ decision keeps its **full** minutes here as one file,
`MTG-NNN-short-slug.md`, written from
[../../09-templates/meeting-minutes-template.md](../../09-templates/meeting-minutes-template.md)
and kept ≤50 lines. The chronological **index** lives in
[../meeting-history.md](../meeting-history.md).

This split exists because minutes are append-only and would otherwise break the 50-line
rule — see [ADR-0002](../adr/ADR-0002-logs-as-index-and-entries.md).

## Records
- [MTG-002-framework-self-audit.md](MTG-002-framework-self-audit.md) — framework self-audit.

## How to Append
1. Copy the minutes template to `MTG-NNN-slug.md` here; fill it; keep it ≤50 lines.
2. Add one index row to [../meeting-history.md](../meeting-history.md).
