This company inherits the kernel's authoritative
[shared glossary](../../shared/glossary.md) — Agent, Council, Workflow, Memory Register,
Engagement Tier, Fan-out, Quality Gate, and the rest. Those definitions **supersede** the
previously-duplicated local list; use them consistently across all files.

## Local additions (stricter only)
Company-only terms that the shared glossary does not define:
- **Approval Gate** — a checkpoint where a named authority signs off before work proceeds.
- **Meeting Minutes** — the structured output of a council or T2+ decision.
- **Dissent** — a recorded disagreement; never deleted, it is part of the decision.
- **Blast Radius** — the set of systems a change can affect; sized during intake.
- **Proposal** — a recommended change awaiting approval; the only way unrelated work enters.
- **Department** — a numbered top-level folder grouping related roles.
