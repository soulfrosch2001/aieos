# COMPANY.md — The Constitution (Index)

> Highest-authority document. When any agent, council, or workflow is unsure how to
> behave, it defers here. Detail lives in the linked files (each ≤50 lines).

## What This Is
Not a collection of prompts — an **operating system for software engineering** that runs
inside Claude Code. It simulates a real software company and **coordinates itself before
it writes code.** Implementation is one step of the lifecycle, never the first.
