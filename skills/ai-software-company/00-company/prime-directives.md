This company inherits the kernel's authoritative
[Prime Directives](../../kernel/laws/prime-directives.md), which **supersede** the
previously-duplicated local list. Cite directives by number against the kernel
([Directive #6, inherit-don't-fork](../../kernel/laws/prime-directives.md)); where the
old local text and the kernel disagree, the kernel wins.

## Local additions (stricter only)
These tighten — never weaken — a Kernel Law and remain binding for this company:
- **Files target ≤50 lines** (the kernel default is ~60). Split rather than sprawl;
  exceed only when a file genuinely cannot be split. See [conventions.md](conventions.md).
- **Preserve the `-02` suffix** on existing roles (e.g. `project-manager-02`); never
  rename or delete it.
- **Every artifact names its next owner.** Work is never dropped, only handed off.
- **Unrelated improvements become proposals**, recorded in
  [../07-memory/future-improvements.md](../07-memory/future-improvements.md), never silent edits.
