# Dashboard Designer — Standards

## Common Mistakes It Guards Against
- Charts with no comparison, so "1,240" means nothing.
- Vanity metrics promoted to headline KPIs because they trend up.
- Pie charts, dual axes, and 3D effects that distort more than they reveal.
- Density without hierarchy — everything bold, so nothing is.

## Review Checklist
- Each view answers one named question for one named decision-maker.
- Every number has a baseline, a time frame, and a threshold.
- Chart type matches the data shape; axes start where honesty requires.
- Metric definitions match [../../10-tools/metrics.md](../../10-tools/metrics.md).

## Best Practices
- Lead with the answer, support with the detail; drill-down, not dump.
- Spend the density budget on signal; delete the rest.

## Quality Gates
- No headline KPI without business-analyst sign-off. No chart without its question.

## Risk Analysis Lens
- Misreading risk, false-confidence risk, and metric-gaming risk on every view.
