# Councils

Status: stable
Type: council-index
Owner: creative-director
Extends: none

## Purpose
The studio's councils are the rooms where significant (T2+) decisions are debated before
anything is built (Directive #3). Each extends a standard-library council by name
(Directive #6) and binds the studio's own chairs to it. Councils are convened, never
standing.

## Councils
- [design-council/](design-council/) — adjudicates core system and world-design decisions. Chair: [creative-director](../01-executive/creative-director/). Extends stdlib [architecture-council](../../councils/architecture-council/).
- [lore-council/](lore-council/) — approves canon additions to the world. Chair: [lore-master](../02-worldbuilding/lore-master/). Extends stdlib [feature-council](../../councils/feature-council/).
