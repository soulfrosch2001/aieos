# 01-executive — Game Balancing Studio Leadership
Status: stable
Type: department
Owner: ceo
Extends: none

The studio's executive spine. Each role binds to a kernel decision-authority level
(see [../../kernel/laws/decision-authority.md](../../kernel/laws/decision-authority.md)).
This studio serves other game, tabletop, and RPG studios — but only through the
Government, never company-to-company ([Directive #5](../../kernel/laws/prime-directives.md)).

| Role | Kernel level | One line |
|------|--------------|----------|
| [ceo](ceo/) | CEO | Sets direction and which titles to balance. |
| [balance-director](balance-director/) | CTO (+ balance-methodology veto) | Owns the balancing methodology and coherence; vetoes unsound or unevidenced changes. |
| [operations-director](operations-director/) | COO | Owns delivery and what ships when; sequences alone. |
| [chief-auditor](chief-auditor/) | Chief Auditor | Quality/process veto; never tunes, never directs. |
| [studio-orchestrator](studio-orchestrator/) | Supreme Orchestrator | Routes balancing briefs, fans out, integrates; never tunes ([Directive #2](../../kernel/laws/prime-directives.md)). |

Every role is a 5-file agent folder per the
[agent contract](../../kernel/contracts/agent.md).
