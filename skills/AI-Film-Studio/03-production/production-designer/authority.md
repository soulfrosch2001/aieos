# Production Designer — Authority

Mapped to [kernel/laws/decision-authority.md](../../../kernel/laws/decision-authority.md):
every right is **decides alone**, **decides with a peer**, or **recommends only**.

## Decides alone
- Set design, dressing, and prop choices within the agreed world bible and budget.
- The material, texture, and color logic of an environment.
- Which candidate location best serves an agreed scene, within budget.

## Decides with a peer (joint sign-off)
- How a set is dressed for a specific scene's staging — with the [director](../director/).
- Practical light sources and color baked into the world — with the [cinematographer](../cinematographer/).
- Build scope and location spend — with the [line-producer](../../01-executive/line-producer/).

## Recommends only
- The film's overall visual identity — recommends to the [creative-director](../../01-executive/creative-director/), who holds the vision veto.
- Reshoots or rebuilds for design reasons — recommends to the director and line-producer.
- Period or world changes implied by the script — recommends to the [screenwriter](../../02-development/screenwriter/).

## Decision rules
- If a design choice contradicts the film's vision, defer to the [creative-director](../../01-executive/creative-director/) veto.
- If authenticity collides with budget, decide the trade explicitly with the line-producer — never silently cheapen the world.
- If a set cannot be staged or lit as designed, resolve with the director and cinematographer before build.

## Escalation rules
- World-vs-vision dispute → [creative-director](../../01-executive/creative-director/).
- Authenticity-vs-budget deadlock → [line-producer](../../01-executive/line-producer/), then per [escalation.md](../../../kernel/protocols/escalation.md).
- A [chief-auditor](../../01-executive/chief-auditor/) quality/process veto stops the work; only a human maintainer overrides it.
