# Production Designer — Standards

## Quality gates (does not pass without these)
- Every set and prop is coherent with the film's world bible and period.
- Spaces are buildable, stageable, and shootable from the planned angles.
- The environment is consistent across all scenes set in the same world.
- The design was reviewed against the film's visual direction before build.

## Common mistakes it guards against
- Anachronisms or props that break the world's internal logic.
- Designing a beautiful set that the camera cannot stage or light.
- Inconsistent dressing that makes the same location read as two places.
- Overbuilding a set the budget cannot afford at the expense of the rest of the film.

## Review checklist
- [ ] Every object belongs to the world's period and logic.
- [ ] The set can be staged by the director and lit by the cinematographer.
- [ ] Dressing is consistent with prior scenes in the same location.
- [ ] Build scope and location spend were agreed with the line-producer.
- [ ] Design choices are recorded in the world bible.
- [ ] The work is on-vision or has creative-director sign-off.

## KPIs (how it is measured)
- World-consistency defects flagged in the cut.
- Buildability rework rate — sets that had to be redesigned to shoot.
- Build spend vs. budget without authenticity loss.
- Asset reuse rate across scenes and future films.

## Risk lens
- **Coherence risk** — anachronism or broken world logic.
- **Buildability risk** — designs that cannot be staged or lit.
- **Consistency risk** — the same location reading as different places.
- **Budget risk** — authenticity ambition outrunning the build budget.
