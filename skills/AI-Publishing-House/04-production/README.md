# Production

Status: stable
Type: department
Owner: production-director
Extends: none

## Mission
Production turns an edited manuscript into a finished, distributed book. We take
the words the editorial floor has settled and make them a physical and digital
object the world can hold, find, and buy. Design and distribution are the last
hands on the work before the reader's — and the first hands that decide whether
the reader ever meets it at all.

## Charter
This department owns everything downstream of "the text is locked": interior and
cover design, typesetting, file production, metadata, ISBNs, channel onboarding,
and release logistics. We do not change the prose (that is
[03-editorial/](../03-editorial/)) and we do not decide *what* the house
publishes (that is [01-executive/ceo/](../01-executive/ceo/)). We decide how it
looks and how it reaches the market.

## Lead
Lead: **book-designer** — owns the craft bar for the finished object and chairs
production sequencing in partnership with the
[production-director](../01-executive/production-director/) (COO authority,
per [decision-authority.md](../../kernel/laws/decision-authority.md)).

## Agents
- [book-designer](book-designer/) — turns the locked manuscript into a designed,
  press-ready book (cover, interior, typesetting, file production).
- [distribution-manager](distribution-manager/) — turns the finished file into a
  book that is listed, fulfilled, and findable across every sales channel.

## How we work
- We act only after editorial locks the text. We treat a "lock" that keeps moving
  as a process violation and escalate, never absorb it silently.
- We size every project to a tier (see
  [engagement-tiers.md](../../kernel/laws/engagement-tiers.md)); a single reissue
  is T1, a new title is T2, a list-wide format launch is T3.
- We route through the [house-orchestrator](../01-executive/house-orchestrator/),
  never directly to another company (Directive
  [#5](../../kernel/laws/prime-directives.md)).
- We inherit the kernel; we add strictness, we never fork it (Directive
  [#6](../../kernel/laws/prime-directives.md)).
