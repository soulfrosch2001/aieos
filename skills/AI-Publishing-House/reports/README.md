The house's reporting surface — health reports and KPI roll-ups that measure AI
Publishing House against the ten AIEOS
[quality dimensions](../../shared/quality-standards.md). Reports are written from the
[report template](../../templates/report.template.md); they observe and recommend —
they never edit or acquire ([Directive #2](../../kernel/laws/prime-directives.md)).

## Domain KPIs tracked
The house's operating measures map onto the ten quality dimensions:
- **Titles signed vs. list target** → Correctness / Consistency.
- **First-pass editorial gate rate** → Correctness / Testability.
- **On-time release rate** → Performance.
- **Errata per 1,000 copies** → Correctness / Security (fail-safe).
- **Memory-update completeness** → Memory hygiene.

See [../../templates/kpi.template.md](../../templates/kpi.template.md) for the KPI
file shape and [../../shared/quality-standards.md](../../shared/quality-standards.md)
for the dimensions every number maps to.
