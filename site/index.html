AIEOS

O sistema operacional para organizações de IA dentro do Claude Code. Ele organiza o trabalho como uma empresa de verdade — roteia, planeja, executa em paralelo, revisa e registra — com um comando só: /aieos.

⬇️ Baixar o AIEOS Ver no GitHub
Requisitos: Windows + Node.js 18+ + Claude Code · grátis e open-source (MIT)

Instalador para Windows

⬇️ Baixar o instalador

Requisitos: Windows + Node.js 18+ + Claude Code · desenvolvedores: código no GitHub

⚠️ Como o app ainda é novo e não está assinado, o Windows pode mostrar “O Windows protegeu o seu PC” — clique em “Mais informações” → “Executar assim mesmo”. É seguro.

Como funciona

1

Instale o Node.js e tenha o Claude Code

Baixe o Node 18+ em nodejs.org e tenha o Claude Code instalado.
2

Rode o instalador do AIEOS

Dois cliques no .exe. Ele instala e se configura sozinho.
3

Digite /aieos no Claude Code

Abra qualquer projeto e descreva o que quer. O AIEOS assume e conduz.

Seus dados

Toda decisão é registrada (sem apagar histórico). Suas conversas viram memória só no seu computador. Compartilhar resumos para melhorar o AIEOS é opcional e desligado por padrão — e, se você ativar, o guarda de segurança remove antes tudo que for sensível. Detalhes na página de privacidade.
