AIEOS

O sistema operacional para organizações de IA dentro do Claude Code. Ele organiza o trabalho como uma empresa de verdade — roteia, planeja, executa em paralelo, revisa e registra — com um comando só: /aieos.

Para quem é

✅ É pra você se…

  • você já usa (ou quer usar) o Claude Code;
  • quer organizar tarefas de IA como uma equipe, não um chat solto;
  • topa instalar o Node.js (1 clique) para rodar.

⛔ Talvez ainda não se…

  • você nunca usou o Claude Code (o AIEOS funciona dentro dele);
  • procura um app "stand-alone" que abre sozinho;
  • não quer instalar pré-requisitos.

Perguntas frequentes

Preciso pagar alguma coisa?

Não. O AIEOS é gratuito e open-source (licença MIT). Você só precisa do Claude Code e do Node.js, que também são gratuitos.

Preciso saber programar?

Para instalar, não — é só rodar o instalador. Para usar bem, ajuda estar confortável com o Claude Code, já que o AIEOS funciona dentro dele.

Funciona no Mac e no Linux?

Tem instalador para Windows (.exe) e macOS (.pkg). No Linux, dá para rodar a partir do código-fonte (clone do repositório).

O Windows mostrou um aviso azul. É vírus?

Não. O aviso aparece porque o app é novo e ainda não tem assinatura digital. Clique em "Mais informações → Executar assim mesmo". O código é aberto e auditável no GitHub.

Ele atualiza sozinho?

Sim. A partir desta versão, novas atualizações chegam automaticamente — você não precisa baixar nada de novo.

Como desligo o compartilhamento de resumos?

Rode aieos memory:share --off a qualquer momento. Veja a página de privacidade para todos os detalhes.

Seus dados

Toda decisão é registrada (sem apagar histórico). Suas conversas viram memória só no seu computador. Compartilhar resumos para melhorar o AIEOS é opcional e desligado por padrão — e, se você ativar, o guarda de segurança remove antes tudo que for sensível. Detalhes na página de privacidade.
