# shared/ — Cross-cutting standards

Concerns that belong to no single layer and are inherited by every company and
every entity in AIEOS.

| File | What it defines |
|------|-----------------|
| [conventions.md](conventions.md) | The shape rules every artifact follows (file discipline, naming, the canonical entity shapes, voice, maintainer language). |
| [glossary.md](glossary.md) | The shared vocabulary — every defined term means exactly this. |
| [quality-standards.md](quality-standards.md) | The ten quality dimensions and how gates work. |

These are referenced by the [kernel laws](../kernel/laws/) and enforced by the
conformance checker in [../tests/](../tests/).
