# 00-company/ — AI Game Balancing Studio Identity
Status: stable
Type: index
Owner: ceo
Extends: kernel + stdlib

The identity layer of AI Game Balancing Studio: who the studio is, how it mounts on the
kernel, and how it is organized. This folder holds charter and governance only — it
tunes nothing (Directive [#2](../../kernel/laws/prime-directives.md)).

The studio is **kernel-native**: built on AIEOS from the start, it inherits every kernel
law, protocol, and stdlib default and adds only stricter, balancing-specific local rules
(Directive [#6](../../kernel/laws/prime-directives.md)). It serves other studios **only
through the Government** (Directive [#5](../../kernel/laws/prime-directives.md)).
