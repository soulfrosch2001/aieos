# Reports
Status: stable
Type: report
Owner: chief-auditor
Extends: none

Point-in-time **health and conformance snapshots** for the AI Marketing Agency.
Reports are written from
[../../templates/report.template.md](../../templates/report.template.md) and read,
never overwritten — a new snapshot is a new dated file or a superseding entry, never
an edit of an old one (Directive [#8](../../kernel/laws/prime-directives.md)).

The [chief-auditor](../01-executive/chief-auditor/) authors these; per the
quality-veto role they **measure and gate, never direct** the work they assess.
