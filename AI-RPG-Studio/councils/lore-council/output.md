# Lore Council — Output

## Output format
Produces **minutes** appended to the studio's [canon](../../memory/registers/canon.md)
register, using the [report.template.md](../../../templates/report.template.md) shape:
Consensus · Remaining concerns · Risks · Alternatives considered · Recommendation ·
Implementation plan · Owners & next steps.

Every set of minutes explicitly records **dissent**: any member overruled by the chair has
their named concern preserved in the entry. Canon is append-mostly (Directive #8) — a fact
that supersedes an earlier one is added as a new, dated entry that cites what it replaces;
the prior entry is never erased.

## Updates to memory
- Always: append the approved canon and recorded dissent to [canon](../../memory/registers/canon.md).
- If debt is taken: note any unresolved canon question and its owner in [campaign-ideas](../../memory/registers/campaign-ideas.md).
- If a lesson emerges: append it to [encounter-log](../../memory/registers/encounter-log.md).
