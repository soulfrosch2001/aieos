# 08-prompts — The Prompts Library

> Reusable instruction blocks agents paste in to run a recurring activity the same
> way every time. Each prompt has Inputs, the Prompt body, and Expected Output.

## Why This Exists
The company repeats a few high-value moves — convening councils, planning,
reviewing, recording memory. These prompts make those moves **consistent and
tier-aware**, so any agent gets the same rigor without reinventing it.

## Index
- [convene-council.md](convene-council.md) — convene a council and produce minutes.
- [plan-feature.md](plan-feature.md) — implementation plan + risks + tests + docs.
- [review-change.md](review-change.md) — five-lens review (correctness, security,
  performance, tests, docs).
- [update-memory.md](update-memory.md) — append the right
  [../07-memory/](../07-memory/) registers after a decision.
