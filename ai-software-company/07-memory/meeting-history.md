# meeting-history.md — Index of Council Minutes

> Corporate Memory: the chronological index of every council and T2+ decision.
> Each entry summarizes a minutes record kept per
> [../09-templates/meeting-minutes-template.md](../09-templates/meeting-minutes-template.md).
> Append-mostly: add meetings as they happen; never delete recorded dissent.

## Entry Format
Columns: `ID | Date | Meeting/Topic | Tier | Chair | Recommendation | ADR/Link`

| ID | Date | Meeting/Topic | Tier | Chair | Recommendation | ADR/Link |
|----|------|---------------|------|-------|----------------|----------|
| MTG-001 | 2026-06-26 | Stand up Corporate Memory | T2 | product-manager-02 | Adopt nine registers | [ADR-0001](adr/ADR-0001-numbered-folder-memory.md) |
| MTG-002 | 2026-06-26 | Framework self-audit | T2 | CTO | Logs→index+entries; fix councils README | [minutes](minutes/MTG-002-framework-self-audit.md) |

## How to Append
Add a new row with the next `MTG-NNN` id once a meeting concludes. Keep the full
minutes (consensus, concerns, risks) in the meeting record; this file is the index.
