AIEOS

O sistema operacional para organizações de IA dentro do Claude Code

👋 Guia de Início Rápido

O que é o AIEOS?

É um sistema que faz o Claude Code trabalhar como uma empresa de verdade: ele recebe seu pedido, planeja, coloca vários agentes trabalhando em paralelo, revisa a qualidade e registra o que aprendeu — tudo com um comando só: /aieos.

Antes de começar, você precisa de 2 coisas

🟢

Node.js 18 ou superior

O AIEOS usa o Node para se configurar. Baixe em nodejs.org (botão verde “LTS”) e instale clicando em Avançar.

🤖

Claude Code

O AIEOS funciona dentro do Claude Code. Se ainda não tem, instale o Claude Code primeiro — é onde você vai digitar o /aieos.

Passo a passo

1

Instale o Node.js

Se ainda não tem, baixe a versão LTS em nodejs.org e instale. (Se já tem, pule esta etapa.)

2

Tenha o Claude Code aberto

Garanta que o Claude Code está instalado e funcionando no seu computador.

3

Rode o instalador do AIEOS

Dê dois cliques em AIEOS-Setup-0.1.0.exe. Ele instala e se configura sozinho (não precisa de administrador).

⚠️ Se o Windows mostrar um aviso azul (“O Windows protegeu o seu PC”), clique em “Mais informações” e depois em “Executar assim mesmo”. Isso acontece porque o app ainda é novo e não está assinado — é seguro.
4

Abra um projeto no Claude Code e use o /aieos

Abra qualquer pasta de projeto no Claude Code e digite o comando. Pronto — o AIEOS assume e conduz a conversa.

claude code — seu projeto

> /aieos faça um resumo do que tem neste projeto
AIEOS: roteando → planejando → executando → relatório…

O que você pode pedir

📋 /aieos revise esta função e sugira melhorias
🚀 /aieos crie um endpoint de cadastro de usuário, com testes
🔎 /aieos explique a arquitetura deste projeto em linguagem simples

É só descrever o que você quer em português — o AIEOS decide o tamanho do trabalho, quem cuida e como entregar.

Atualizações & sua memória

🔄 O AIEOS se atualiza sozinho. Quando sai uma versão nova, ela chega automaticamente — você não precisa reinstalar nem fazer nada.

🧠 Suas conversas viram memória só no seu computador. Nada é enviado para fora sem o guarda de segurança ocultar antes o que for sensível (senhas, chaves, etc.).

É seguro?

Sim. Toda decisão é registrada (sem apagar histórico), segredos como senhas e chaves são ocultados automaticamente, e conteúdo perigoso é colocado em quarentena antes de qualquer coisa ser compartilhada. Para desinstalar a qualquer momento: Adicionar ou Remover Programas → AIEOS → Desinstalar.
