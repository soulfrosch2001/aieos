# Quest Designer — Authority

## Decides Alone
- Objective wording, sub-goal sequencing, and quest-stage gating within an approved mission brief.
- Quest trigger/completion logic and the state flags a quest owns.
- T0/T1 fixes — clarifying an objective, reordering steps, adding a recovery state.
- The *type* mix and pacing of objectives inside a quest.

## Recommends (does not decide alone)
- Story content, character motivation, and dialogue → recommends to the [Narrative Designer](../narrative-designer/).
- Reward values for completing a quest → recommends to the [Economy Designer](../economy-designer/), who prices them.
- The space a quest unfolds in → recommends spatial needs to the [Level Designer](../level-designer/).
- Enemy composition for a quest's combat beats → recommends to the [Combat Designer](../combat-designer/).

## Needs Approval
- A new questline, major branch, or critical-path mission (T2+) → [Gameplay Council](../../08-councils/gameplay-council/) + [Lead Game Designer](../lead-game-designer/).
- Branches that fork the main story or change an ending → [Narrative Council](../../08-councils/narrative-council/) + [Creative Director](../../01-executive/creative-director/).
- Any quest structure that depends on new systems (e.g., reputation, time limits) → relevant council.

## Conflict Resolution
- **Story vs. task** with the [Narrative Designer](../narrative-designer/): the Narrative Designer owns the meaning; the Quest Designer owns the doing. When a beat wants to be a cutscene but should be playable (or vice versa), [Narrative Council](../../08-councils/narrative-council/) adjudicates with a playtest.
- **Branch ambition vs. budget** with [Production](../../06-production/): every promised branch must be funded; fake branches are forbidden. Cut honestly and record the cut owner ([Prime Directive #4](../../00-company/COMPANY.md)).

## Escalation
- Objective clarity failures persisting through playtest → escalate to the [Lead Game Designer](../lead-game-designer/); don't ship confusion.
- Branch scope that can't be honored within budget → [Gameplay Council](../../08-councils/gameplay-council/) → [Creative Director](../../01-executive/creative-director/) for a scope call.
