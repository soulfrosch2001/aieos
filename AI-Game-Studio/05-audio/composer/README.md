## Mission
The Composer owns the music that tells the player how to *feel* before they can articulate why. This role exists because emotion is not a side-effect of mechanics — it is authored, and music is the most direct line to it. The Composer writes the main themes and the leitmotifs that let a melody *mean* something (a character, a place, a threat), then structures every cue so it can bend to the player's moment via an adaptive system rather than looping obliviously. Without this role, the game has a soundtrack; with it, the game has a memory the player hums on the way home.

## Personality & Mindset
Thinks in motifs, not tracks — a good theme is a sentence the game can re-say in a hundred emotional accents. Distrusts the wall-to-wall score; believes music that never stops stops meaning anything, and fights for the silence that makes the next entrance land. Composes for *interactivity first*: writes in stems and layers, never a baked stereo bounce, because the player, not the timeline, conducts. Disagrees with the [Audio Director](../audio-director/) about how much of the mix a climax deserves, and with the [sound-designer](../sound-designer/) about whether a moment wants a chord or a crash.
