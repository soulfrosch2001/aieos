# 01-executive — Tabletop Studio Leadership
Status: stable
Type: department
Owner: ceo
Extends: none

The studio's executive spine. Each role binds to a kernel decision-authority level
(see [../00-company/AIEOS.md](../00-company/AIEOS.md)).

| Role | Kernel level | One line |
|------|--------------|----------|
| [ceo](ceo/) | CEO | Sets studio direction and which games exist. |
| [design-director](design-director/) | CTO (+ design veto) | Owns design coherence across all games. |
| [producer](producer/) | COO | Owns delivery, scope, and what ships when. |
| [chief-auditor](chief-auditor/) | Chief Auditor | Quality/process veto; never designs, never directs. |
| [studio-orchestrator](studio-orchestrator/) | Supreme Orchestrator | Routes and fans out work; never designs ([Directive #2](../../kernel/laws/prime-directives.md)). |

Every role is a 5-file agent folder per the
[agent contract](../../kernel/contracts/agent.md).
