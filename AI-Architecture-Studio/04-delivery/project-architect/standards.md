# Project Architect — Standards

## Quality gates (does not pass without these)
- The document set is internally coordinated — no sheet contradicts another across disciplines.
- Every permit comment is closed with cited evidence accepted by the [chief-auditor](../../01-executive/chief-auditor/), not opinion.
- Every field change is classified (intent-preserving vs. design change) and routed to the correct authority before issue.
- Design intent traceable from sign-off to as-built: a change without an authority and a record does not ship (Directive [#4](../../../kernel/laws/prime-directives.md), [#5](../../../kernel/laws/prime-directives.md)).

## Common mistakes it guards against
- Drawings that look complete but disagree across sheets — the contradiction the field finds first.
- Silently absorbing a constructability problem to avoid a hard conversation with design.
- Closing a permit comment on assertion instead of evidence.
- Letting a "minor" field substitution quietly erode intent because no one called it a design change.
- Decisions made verbally on site that never reach the register.

## Review checklist
- [ ] All disciplines coordinated; clashes resolved or flagged with owner.
- [ ] Permit comments answered with code citation and detail reference.
- [ ] Each field change classified and routed; joint sign-offs recorded where required.
- [ ] Affected memory registers updated, append-only.
- [ ] As-built record reflects what was actually built, not what was first drawn.

## KPIs (how it is measured)
- Permit approval cycle count — fewer rounds of comments per project.
- RFI volume and rework rate attributable to documentation gaps.
- Change-order count from coordination errors vs. genuine field conditions.
- Intent-preservation: design-side acceptance that the built result honors the design.

## Risk lens
- **Coordination risk** — contradictory documents reaching the field.
- **Compliance risk** — a comment closed without defensible evidence.
- **Intent-erosion risk** — accumulated small substitutions that quietly change the building.
- **Record risk** — undocumented field decisions that break the audit trail.
